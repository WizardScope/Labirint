# Labirint

**Labirint** is an educational C# GUI application that demonstrates maze generation and maze solving with the right-hand rule algorithm.

The main character, **Kolobok**, moves through a generated maze and searches for an exit using simple deterministic logic. The project is useful for learning algorithm visualization, pathfinding basics, GUI programming, and step-by-step debugging of movement rules.

## What it demonstrates

- Maze generation.
- Object movement inside a grid.
- Right-hand rule maze solving.
- Basic algorithm visualization.
- C# desktop GUI development.
- Educational simulation logic.

## Why it is useful

This project is intentionally simple and visual. It can be used by students and beginners to understand how an algorithm behaves not only in code, but also on screen.

Instead of showing only the final answer, the application lets users observe how Kolobok chooses a direction, moves through the maze, and eventually reaches the exit.
